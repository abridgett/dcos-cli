Get and set DCOS CLI configuration properties

Usage:
    dcos config --info
    dcos config append <name> <value>
    dcos config prepend <name> <value>
    dcos config set <name> <value>
    dcos config unset [--index=<index>] <name>
    dcos config show [<name>]
    dcos config validate

Options:
    -h, --help       Show this screen.
    --info           Show a short description of this subcommand.
    --version        Show version.
    --index=<index>  Position in the list of the desired element. The first
                     element in the list has an index of zero.

Positional Arguments:
    <name>           The name of the property.
    <value>          The value of the property.

Subcommands:

    append           Add a value to the end of a list. Items earlier in the
                     list take precedence over items that follow. For
                     example, you can append a local package repository to the
                     packages.sources list so that it is searched after the
                     Mesosphere package repository.

    prepend          Add a value to the beginning of a list. Items earlier in
                     the list take precedence over items that follow. For
                     example, you can prepend a local package repository to the
                     packages.sources list so that it is searched before the
                     Mesosphere package repository.

    set <name> <value>

                     Add a parameter to the DCOS configuration file.

    unset [--index=<index>] <name>

                     Remove a parameter from the DCOS configuration file.

    show [<name>]    Display the contents of the DCOS configuration file,
                     or optionally, of the specified parameter.

    validate         Validate changes made to the configuration file. You
                     should run this command every time you make
                     configuration changes.
